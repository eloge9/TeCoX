# TeCoX - Tech Community Extensible

TeCoX est une plateforme web pour une **communauté de développeurs**.  
Les utilisateurs peuvent consulter des pages publiques, créer et commenter des posts, participer à des événements.  

## Fonctionnalités
- Visiteurs : pages publiques
- Membres : posts, événements, profil
- Admin : gestion des contenus et utilisateurs
- Super Admin : gestion des admins et paramètres globaux

## Installation rapide
1. Cloner le projet : `git clone <URL>`
2. Créer un environnement : `python -m venv envTeCoX`
3. Activer l'environnement et installer les dépendances : `pip install -r requirements.txt`
4. Configurer la base de données dans `settings_local.py`
5. Lancer le serveur : `python manage.py runserver`
